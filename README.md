# The RAD Game

This is a type of a memory game, where you pass from one room to another via the available doors in each room, and your goal is to open the final golden door.

In this game, you compete against five random players in a race against time to reach the final door. Use keyboard arrows to move within rooms, and press enter to walk through doors, leading to an adjacent room. You can also go back to the previous room via the concentric circles.

## NOTE

The game is implemented in Python. It is similar to the Rooms & Doors game, also found in my GitHub repository, which is an extended online version with more complexity and better graphics.
